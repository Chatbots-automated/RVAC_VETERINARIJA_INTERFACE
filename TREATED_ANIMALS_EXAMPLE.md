## Key Features in Visual Display

### 1. Eil. Nr. (Row Number) - Column 1
```
┌──────┐
│  1   │  ← Yellow background
│      │  ← Bold font
│      │  ← Sequential numbering
└──────┘
```

**CRITICAL:** This number cannot be changed once report is generated!

### 2. Registration Date - Column 2
```
┌────────────┐
│ 2024-01-15 │  ← Lithuanian date format (YYYY-MM-DD)
└────────────┘
```

### 3. Owner Details - Column 3
```
┌───────────────────────┐
│ UAB "Pieno ūkis"      │  ← Name/Company (bold)
│ Kaimo g. 1, Miestas   │  ← Address (smaller text)
└───────────────────────┘
```

### 4. Species & Sex - Column 4
```
┌────────────┐
│ Bovine     │  ← Species
│ Female     │  ← Sex (if available)
└────────────┘
```

### 5. Age - Column 5
```
┌────────┐
│ 2 m.   │  ← 2 years 3 months
│ 3 mėn. │
└────────┘

OR

┌────────┐
│ 6 mėn. │  ← 6 months
└────────┘

OR

┌────────┐
│ 1 m.   │  ← 1 year
└────────┘
```

**Calculated automatically** from `age_months` or `birth_date`!

### 6. Animal Tag - Column 6
```
┌────────────┐
│LT123456789 │  ← Bold, centered
└────────────┘
```

### 7. First Symptoms Date - Column 7
```
┌────────────┐
│ 2024-01-10 │  ← When symptoms were first noticed
└────────────┘
```

### 8. Animal Condition - Column 8
```
┌──────────┐
│ Apatiniu │  ← Free text description
└──────────┘
```

### 9. Tests Performed - Column 9
```
┌────────────┐
│ Kraujo     │  ← Blood test
│ tyrimas    │
└────────────┘

OR

┌────────────┐
│ -          │  ← No tests performed
└────────────┘
```

### 10. Clinical Diagnosis - Column 10
```
┌──────────────────┐
│ Mastitas         │  ← Disease name (bold)
│ Ūminis mastitas  │  ← Additional diagnosis details (if different)
└──────────────────┘
```

### 11. Veterinary Services - Column 11
```
┌───────────────────────────────┐
│ Gydymas                       │  ← Service description
│ 💊 PENSTREP 400, MELOXICAM    │  ← Medications used (auto-collected)
└───────────────────────────────┘
```

**Medications automatically collected** from treatment records!

### 12. Withdrawal Period - Column 12
```
┌──────────────┐
│ 🥩 2024-01-22│  ← Meat withdrawal (red)
│ 🥛 2024-01-20│  ← Milk withdrawal (blue)
└──────────────┘

OR

┌──────────────┐
│ -            │  ← No withdrawal period
└──────────────┘
```

### 13. Outcome - Column 13
```
┌──────────┐
│ Pasveiko │  ← Recovered
└──────────┘

OR

┌──────────┐
│ Gydomas  │  ← Under treatment
└──────────┘
```

### 14. Veterinarian - Column 14
```
┌───────────────────┐
│ ARTŪRAS ABROMAITIS│  ← Vet name (bold)
└───────────────────┘
```

## Data Flow Example

```
1. User creates treatment:
   ├─ Select animal (LT123456789)
   ├─ Enter diagnosis (Mastitas)
   ├─ Add medicines (PENSTREP 400, MELOXICAM)
   ├─ Set withdrawal periods
   └─ Complete treatment
        ↓
2. System automatically captures:
   ├─ Registration date: today
   ├─ Owner: from animal record
   ├─ Species/sex: from animal record
   ├─ Age: calculated from birth_date or age_months
   ├─ Tag: from animal record
   ├─ First symptoms: if entered
   ├─ Condition: if entered
   ├─ Tests: if entered
   ├─ Diagnosis: selected disease
   ├─ Services: if entered
   ├─ Medications: auto-collected
   ├─ Withdrawal: calculated
   ├─ Outcome: if entered
   └─ Veterinarian: from user profile
        ↓
3. Report shows in journal:
   ├─ Eil. Nr.: 1 (sequential)
   └─ All 14 columns populated ✅
```
